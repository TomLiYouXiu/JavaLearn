哥斯达黎加计划强化渔业资源保护力度

中国新闻网 新浪新闻 (11人参与分享) 收藏本文 2018-10-31 16:55

中新网10月31日电 据南美侨报网报道，哥斯达黎加渔业联盟(Federación Costarricense de Pesca，简称Fecop)近日通过推出“哥斯达黎加人的金枪鱼(Atún para los ticos)”活动，呼吁哥斯达黎加政府、渔业主管部门和民众保护以金枪鱼为主的国家海洋渔业资源。

哥斯达黎加“elmundo.cr”10月24日报道，“公路广告牌上、社交媒体和网页中都可以看到保护鱼类资源的宣传标语，我们还发起了请愿书签名活动，希望在让公众了解哥斯达黎加海洋渔业资源现状的同时，向国家主管部门表达我们的担心。”哥斯达黎加渔业联盟项目协调员亨利•马林(Henry Marín)说。

“据不完全统计，国际捕捞船队在哥斯达黎加领海内捕获的金枪鱼中，有26%为非法渔获，既没有向我国有关部门报告，也从不从我国港口经过，这侵害了我国人民的利益。”马林说。

“哥斯达黎加渔民捕获的金枪鱼数量时多时少。消费者和市场对于除了金枪鱼以外的其他鱼类都有比较强的保护意识，并愿意在价格方面支持那些具有可持续性的渔业活动。想要获得同等地位，金枪鱼的资源必须充足，而哥斯达黎加海域中的金枪鱼目前正在遭受外国非法船队的掠夺。”马林补充道。

哥斯达黎加渔业联盟最近在蓬塔雷纳斯省(Puntarenas)和瓜纳卡斯特省(Guanacaste)会见了当地的渔民和职业钓鱼运动员代表，旨在与当地社区成员更加密切地讨论金枪鱼渔业资源的保护问题。

另外，哥斯达黎加渔业联盟还正在全国范围内推广名为“绿杆(Greenstick，指在船上斜立纤维玻璃杆，杆顶距水面30英尺，并在杆上悬挂人工鱿鱼诱饵来捕捞金枪鱼)”的捕鱼技术。据了解，该商业捕鱼方法能够在捕获金枪鱼时，避免误捕其他鱼类，因此在几年前开始被一些勇于尝试的哥斯达黎加渔民采所用。该技术的开发得到了哥斯达黎加国家培训学院(Instituto Nacional de Aprendizaje，简称INA) 和哥斯达黎加渔业和水产养殖研究所(INCOPESCA)的支持，并于今年获得了官方技术认证。

2017年，哥斯达黎加渔业联盟成功说服政府，将外国金枪鱼围网船队的许可数量从43个减少到13个，有效地提高了对马林鱼、鲨鱼、海龟、鲷鱼，以及其他多个兼捕物种的保护力度。
